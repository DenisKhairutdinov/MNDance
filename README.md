**[MNDance](https://deniskhairutdinov.github.io/MNDance/)**

MNDance - лендинг студии танцев, для которого я самостоятельно
разработал дизайн и реализовал его в коде. Проект создавался с упором на современный
интерфейс, плавные анимации и удобную навигацию по странице. Исходный файл дизайна не
публикуется, так как проект создавался в процессе разработки и использовался как рабочий
инструмент.

## Пользовательский интерфейс
- Навигация по секциям страницы;
- Автоматическое определение активного раздела меню;
- Интерактивное расписание занятий;
- Модальные окна на основе нативного элемента dialog;
- Собственная реализация бесконечного слайдера;
- Плавная прокрутка страницы;
- Анимации появления элементов при скролле;

## JavaScript / TypeScript
- Модульная архитектура проекта;
- Использование паттерна Pub/Sub для взаимодействия между независимыми модулями;
- Работа с DOM через отдельные модули;
- Обработка пользовательских событий;

## Технологии
HTML5 • SCSS • LightningCSS • JavaScript (ES6+) • TypeScript • Vite • Vituum • Nunjucks • GSAP • ESLint • Stylelint • Prettier

## Lighthouse
Использование ScrollSmoother положительно влияет на пользовательский опыт и качество анимаций,
однако снижает оценку Lighthouse Performance на десктопных устройствах. После анализа было
принято решение сохранить данное решение, т.к. сайт остается стабильным, не имеет заметных
проблем с отрисовкой интерфейса и корректно работает на мобильных и десктопных устройствах.
